# Data-Cleaning-Visualization
This project, my team members and I are tasked with picking independent variables from the UNSW-NB15 dataset and comparing them to a dependent variable using R.

The objective of the project is to use independent variables, which I have chosen to use 6, and compare it to how it affects the dependent variable which is attac_category that is chosen for this project. I have decided to use K-Means Clustering due to memory limitation of my system and use 20 clusters so that it is memory-efficient. Not only that, but I have also implemented Random Forest for prediction modeling, which also displays a confusion matrix and shows how accurate it is in predicting whether it is an attack or not.
